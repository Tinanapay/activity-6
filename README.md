# CS Elective Frontend Activity

This project is a requirement for a CS Elective (frontend development)

##  Description
This application demonstrates a basic user authentication flow using a React frontend and a PHP backend. It includes user registration, login, and a protected home page.

##  Features
- User Registration with frontend password validation
- User Login system
- Protected Home page (accessible only when logged in)
- Logout functionality
- Basic routing using React Router

##  Technologies Used
- React (Vite)
- JavaScript (JSX)
- PHP (backend)
- HTML & CSS
- LocalStorage (for simple authentication state)
